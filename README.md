# Scene 3D rendering engine featuring ray tracing algorithm

```sh
# Build.
$ make

# Display help.
$ ./dist/main --help

# Render example scene using scattering technique with five sampels per pixel.
$ ./dist/main --scattering 1 --antialiasing 5
```
